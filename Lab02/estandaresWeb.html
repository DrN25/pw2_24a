ESTANDARES WEB

En esta seccion encontraras informacion acerca de la W3C y algunos

de los mas importantes estandares web como SVG, WOFF, WebRTC y XML.

  • W3C(Worl Wide Web Consortium)

  • Es una organizacion internacionales que publica y supervisa estandares para las tecnologias relacionadas a la web, como HTML, CSS y JavaScript.

  • SVG(Scalable Vector Graphics)

  • Es un formato basado en XML para graficos vectoriales bidimensionales, permitiendo describir graficos e imagenes con elementos geometricos y otros componentes graficos.

  • WOFF (Web Open Font Format)

  • Utilizado para empaquetar fuentes que se utilizan en paginas web, brindando una mayor seguirdad y eficiencia en el uso de fuentes personalizadas.

  • WebRTC (Web Real-Time Communication)

  • Es una tecnologia que permite la comunicacion en tiempo real entre navegadores web y dispositivos, logrando las transmisiones de datos eficiente.

  • XML (Extensible Markup Language)

  • Es un formato de marcado que permite la estructura de datos y la portabilidad entre diferentes sitemas y apliaciones. Es interpretado por diversos lenguajes y plataformas.
